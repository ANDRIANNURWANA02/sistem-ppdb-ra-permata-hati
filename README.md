# Sistem PPDB RA Permata Hati  Aplikasi pendaftaran siswa baru berbasis web menggunakan Laravel.  
## Fitur - Login &amp; Register - Dashboard User &amp; Admin - Pendaftaran Siswa - Manajemen Data - Export Data Pendaftarn  
## Instalasi 1. git clone 2. composer install 3. cp .env.example .env 4. php artisan key:generate 5. php artisan migrate
